# Commandes pour Pousser vers GitHub

## Si vous avez un Personal Access Token GitHub

```bash
cd "/Users/sabrikhalfallah/Downloads/BTP SMART PRO"
git push https://VOTRE_TOKEN@github.com/svbkhl/btp-smart-pro.git main
```

**Remplacez `VOTRE_TOKEN`** par votre Personal Access Token GitHub.

---

## Pour créer un Personal Access Token

1. Allez sur : https://github.com/settings/tokens
2. Cliquez sur "Generate new token (classic)"
3. Donnez un nom : `Vercel Deploy`
4. Cochez `repo` (toutes les cases)
5. Cliquez sur "Generate token"
6. **COPIEZ LE TOKEN** (vous ne le reverrez plus !)

---
